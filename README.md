# 🌦️ Weather & Air Quality Dashboard

An interactive dashboard that combines **weather forecasting** and **air quality monitoring** for Indian cities like **Shillong, Varanasi, and Noida**. It helps users understand environmental conditions using clean visuals and actionable insights.

## 📌 Project Title
**Weather & Air Quality Forecast Dashboard**

## 📖 Short Description / Purpose
This project was built to analyze and visualize **current weather**, **weekly forecast**, and **air quality conditions** for multiple cities in one place.  
Its purpose is to help users make better decisions related to **travel, health, and daily outdoor activities**.

## 🛠️ Tech Stack
- **Power BI** – dashboard development and visualization
- **Excel / CSV** – data collection and preprocessing
- **DAX** – KPI calculations and logic
- **Power Query** – data cleaning and transformation

## 📂 Data Source
The dashboard uses structured weather and AQI data containing:
- Temperature (current, min, max)
- Weather condition
- Visibility, humidity, precipitation
- Pressure, UV index, wind speed
- Sunrise and sunset timings
- 7-day forecast
- AQI and pollutants (PM10, PM2.5, CO, NO2, SO2, O3)
- Rain probability

**Source:** Weather API / public weather dataset / environmental data file

## ✨ Features
- Current weather overview
- Weekly temperature forecast
- Air Quality Index (AQI) monitoring
- Pollutant-level analysis
- Sunrise and sunset timings
- Rain probability tracking
- Humidity, pressure, wind, visibility, and UV monitoring
- Multi-city comparison
- Interactive visual dashboard

## 🧠 Example / Use Case
A user can compare **Shillong, Noida, and Varanasi** to check:
- Which city has better weather
- Which location has safer AQI
- Which day has a lower chance of rain
- Which city is better for outdoor activity or travel

## ❓ Key Questions Answered
- Which city has the best weather conditions?
- Which city has the worst air quality?
- How does the temperature change over the next 7 days?
- Which city has the highest rainfall probability?
- What is the AQI health category for each city?
- Which day is best for travel or outdoor work?
- How do pollutant levels vary across cities?

## 📈 Business Impact and Insights
This dashboard provides useful insights for:
- Travel planning
- Public health awareness
- Environmental monitoring
- Smart city reporting
- Daily commute and event planning

### Key Insights
- **Shillong** shows comparatively cleaner air and healthier AQI.
- **Noida** reflects **very unhealthy AQI**, indicating high pollution.
- **Varanasi** falls into **unhealthy for sensitive groups**.
- Forecast and rain probability help identify safer and more suitable days.

## 📊 Dashboard Sections
- **Current Weather Overview** – temperature, condition, min/max
- **Weekly Forecast** – 7-day forecast cards and trend chart
- **Atmospheric Metrics** – humidity, pressure, visibility, UV, wind
- **Air Quality Monitoring** – AQI score and pollutant indicators
- **Rain Probability** – daily chance of rain

## 🚀 How to Use
1. Open the Power BI dashboard file
2. Select a city or compare city cards
3. Review current weather and AQI
4. Analyze weekly forecast and rain chances
5. Use insights for travel, health, and planning

The final output is a **clean, modern, and informative weather intelligence dashboard** that converts raw weather and AQI data into meaningful visual insights.
