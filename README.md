# 🌦️ Weather & Air Quality Dashboard

An interactive **Weather Forecast and Air Quality Dashboard** built to provide real-time and forecast-based environmental insights for multiple Indian cities such as **Shillong, Varanasi, and Noida**.  

This dashboard helps users quickly understand **temperature trends, rainfall chances, air quality, humidity, visibility, wind speed, sunrise/sunset timing, and other atmospheric conditions** through an intuitive and visually engaging interface.

---

## 📌 Project Title

**Weather & Air Quality Forecast Dashboard**

---

## 📖 Short Description / Purpose

This project was developed to create a **city-wise weather monitoring dashboard** that combines **forecast weather data** with **air quality insights** in a single user-friendly interface.

The main purpose of this project is to help users:
- Track current and upcoming weather conditions
- Monitor air pollution levels
- Compare multiple cities
- Make better decisions for travel, health, and daily planning

This dashboard transforms raw environmental data into meaningful visual insights for easier interpretation.

---

## 🛠️ Tech Stack

The following technologies and tools were used in this project:

- **Power BI** – Dashboard creation and data visualization
- **Excel / CSV** – Data storage and preprocessing
- **DAX** – Calculated measures and KPI logic
- **Power Query** – Data transformation and cleaning
- **Weather Dataset / API Exported Data** – Forecast and AQI information
